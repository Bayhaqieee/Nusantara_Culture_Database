## Setup

1. Clone the repository:
    ```bash
    git clone 
    ```

2. Navigate to the project directory:
    ```bash
    cd dashboard
    ```
3. Install all related Requirement:
    ```bash
    pip install -r requirements.txt
    ```
4. Run and Save the model by the name:
    ```bash
    thechosen.ipynb
    ```
